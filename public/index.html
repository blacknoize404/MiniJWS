MiniJWS Demo

Lightweight modular HTTP server — Java 25

Try these routes

  • /hello — plain text
  • /hello?name=MiniJWS — with query param
  • /hello/World — path parameter
  • /api/info — JSON API
  • /echo — echo request details
  • /set-cookie — set a cookie
  • /get-cookies — read cookies
  • /old-path — 301 redirect

Features

Middleware CORS Path Params Static Files Keep-Alive Gzip Rate Limit Cookies Redirect Logging
